========================================
   999 PRO TERMINAL v1.0
   Система приема заказов
========================================

ОПИСАНИЕ
--------
Программа превращает компьютер в терминал
для приема файлов от клиентов по Wi-Fi.

========================================
СИСТЕМНЫЕ ТРЕБОВАНИЯ
========================================
- Windows 10/11
- Python 3.11 или выше
- Wi-Fi роутер (для подключения клиентов)

========================================
ЗАПУСК
========================================

1. Убедитесь, что установлен Python 3.11+
   (проверка: python --version)

2. Дважды кликните на run.bat

3. Программа автоматически:
   - Установит зависимости
   - Определит локальный IP
   - Сгенерирует QR-код
   - Откроет панель оператора

========================================
ИНТЕРФЕЙСЫ
========================================

ПАНЕЛЬ ОПЕРАТОРА (открывается автоматически):
  http://127.0.0.1:8000/panel
  
  Функции:
  - QR-код для клиентов
  - Список входящих заказов
  - Поиск по имени/ID
  - Открытие папки заказа
  - Удаление заказов
  - Звуковое оповещение

СТРАНИЦА КЛИЕНТА (только через QR):
  http://LOCAL_IP:8000/send
  
  Функции:
  - Ввод имени
  - Отправка файлов
  - Номер заказа

========================================
РАБОТА С КЛИЕНТАМИ
========================================

1. Клиент подключается к Wi-Fi
2. Наводит камеру на QR-код
3. Открывается страница отправки
4. Вводит имя
5. Выбирает файлы
6. Получает номер заказа

========================================
ХРАНЕНИЕ ФАЙЛОВ
========================================

Структура папок:
uploads/
  YYYY-MM-DD/        (дата заказа)
    ИМЯ_ID/          (имя клиента + ID)
      файлы...

Пример:
uploads/
  2024-03-15/
    Иван_A1B2C3/
      document.pdf
      photo.jpg

========================================
ПЕРЕКЛЮЧЕНИЕ ТЕМЫ
========================================

Нажмите на кнопку "День/Ночь" 
в правом верхнем углу для переключения
между темной и светлой темой.

========================================
РЕШЕНИЕ ПРОБЛЕМ
========================================

QR-код не открывается:
- Проверьте, что клиент в той же Wi-Fi сети
- Проверьте брандмауэр Windows
  (разрешите Python на порту 8000)

Файлы не отправляются:
- Проверьте размер файлов
- Перезапустите программу

Нет звука уведомления:
- Проверьте громкость
- Кликните на страницу (браузеры блокируют
  автовоспроизведение звука)

========================================
ТЕХНИЧЕСКАЯ ИНФОРМАЦИЯ
========================================

Порт: 8000
Протокол: HTTP + WebSocket
База данных: JSON файл (orders.json)
